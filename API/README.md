TO RUN THE CODE:

STEP 1: Install dependecies
````
pip3 install -r requirements.txt

````

STEP 2: Run the main.py
````
python3 main.py

````

STEP 3: RUN FASTAPI
```
uvicorn main:app --reload

````

Uvicorn will run on http://127.0.0.1:8000
